Calculá el valor del envío de tus discos

Este simulador calcula el valor de tu envío según las dimensiones, peso y monto del pedido. En caso de que el pedido supere los $20.000.- el envio será gratis. Por el momento hay una tarifa plana de $2000, para todos los envíos, pero la intensión de este simulador es que varíe segun las caracteristicas del envío (peso, dimensiones, codigo postal y tipo de envio -domicilio o sucursal.

Se utilizó un condicional, con if y else, para indicar el valor del envio, o envio gratis, segun el monto introducido por el usuario. Además con do while se colocó un bucle indicando que el valor indicado no es valido, cada vez que no se coloque un parseInt en el prompt, y vuelva a solicitar el valor del envío. Una vez que el numero colocado es un parseInt, a través de return el alert devuelve valor de envio, o envio gratis.

Todo esto está dentro de una función llamada "calcularEnvio"
